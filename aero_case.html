Case Study 1

Analysis and optimisation of wind turbine blade computational models

A research organisation in mainland Europe working on off-shore wind turbines had been looking at methods of optimising wind turbine shape designs for large 5MW output wind turbines near coastal areas.

The task consisted of creating both a numerical as well as computational model of a small 2-bladed wind turbine used for research purposes. This was to validate the analysis and computational methodology development using a much larger NREL (National Renewable Energy Laboratory) offshore 5-MW baseline wind turbine. The wind turbine was modelled and meshed using the software package Rhino 3D. Solving and post-processing was done using Star CCM++. Both steady state and transient analyses were done in a defined control volume representing a wind tunnel.

Result: Computational models prepared using the baseline wind turbines can now be used to also analyse other wind turbine models with ease as model has been validated with published and numerical data.
